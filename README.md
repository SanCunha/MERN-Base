## MERN Stack Base (Using MongoDB with Docker)

##### Versão em Português Abaixo

###### [Se você é dev aprender inglês deve ser uma prioridade ;)]

### How to use?

Download the directory, go to the database folder, and use the command below to start a container with a MongoDB instance.

```bash
docker-compose up -d
```

Then, repeat the command below in the root of the project, inside client, and inside server. This will install all the dependencies.

```bash
npm i
```

To start everything, return to the root and use:

```bash
npm run dev
```

## Base MERN Stack (Usando MongoDB com Docker)

### Como usar?

Faça o download do diretório, vá na pasta database e utilize o comando abaixo para iniciar um container contendo uma instância MongoDB.

```bash
docker-compose up -d
```

Depois repita o comando abaixo na raíz do projeto, dentro de clien e dentro de server. Assim todas as dependências serão instaladas.

```bash
npm i
```

Para iniciar tudo volte na raíz e utilize:

```bash
npm run dev
```
